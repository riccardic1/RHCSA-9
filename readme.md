# RHCSA (RHEL 9) Study Notes

This repository contains my personal study notes, command references, practice exercises, and configuration examples while preparing for the **Red Hat Certified System Administrator (RHCSA) - RHEL 9** exam.

## 📘 About
The goal of this repository is to organize and reinforce key topics required for the RHCSA certification. These notes are structured around real-world system administration tasks and the official Red Hat exam objectives for RHEL 9.

## ✅ Topics Covered
- Essential tools & command-line environment
- User and group management
- File permissions, ACLs, and SELinux
- Storage management (LVM, partitions, VDO, Stratis)
- System services and systemd
- Networking configuration
- Firewalld & security basics
- Boot process and kernel interaction
- Containers (Podman)
- Scripting and automation fundamentals

## 🧠 Study Approach
These notes focus on:
- Hands-on practice
- Practical configuration examples
- Scenario-based learning
- Rapid recall of key commands and workflows

## 📂 Repository Structure
```
notes/           # Topic-based study notes
commands/        # Useful command references
labs/            # Hands-on lab exercises
configs/         # Example configuration files
checklists/      # Exam prep checklists and quick references
```

## 🛠️ Recommended Lab Environment
- RHEL 9 or AlmaLinux 9
- At least two virtual machines (for networking, user, and storage scenarios)
- Podman installed for container-related objectives
